About the PAU Dress Code Compliance System

Project Overview

This AI-powered dress code compliance system was developed as a Computer Science final year project to help Pan-Atlantic University female students verify their outfit compliance before leaving their hostels.

Project Goals
  • Reduce subjective enforcement of dress codes
  • Provide real-time feedback to students
  • Prevent academic disruptions due to dress code violations
  • Enhance student confidence and reduce embarrassment
  • Create an unbiased, consistent evaluation system
Key Features
  • AI-powered image analysis using ResNet architecture
  • Real-time compliance checking
  • Specific violation detection and feedback
  • Mobile-friendly responsive design
  • Privacy-focused (images not permanently stored)
Technical Implementation
Backend Technology
  • Django 4.2.7
  • PyTorch
  • OpenCV
  • PIL/Pillow
  • SQLite Database
Frontend Technology
  • HTML5 & CSS3
  • Bootstrap 5
  • JavaScript (ES6)
  • Font Awesome Icons
  • Responsive Design
AI/ML Technology
  • ResNet-50 Architecture
  • Transfer Learning
  • Custom Dataset Training
  • Google Colab Training
  • Binary Classification
Development Process
Data Collection & Training
  • Dataset Creation: Custom dataset of PAU female student outfits
  • Labeling: Manual annotation of compliant vs non-compliant images
  • Training Environment: Google Colab with GPU acceleration
  • Model Fine-tuning: ResNet-50 pre-trained on ImageNet
  • Validation: Cross-validation and performance testing
Development Environment
  • IDE: Visual Studio Code
  • Version Control: Git
  • Backend Framework: Django
  • Model Training: Google Colab
  • Testing: Local development server
Expected Impact
Benefits for Students
  • Reduced anxiety about dress code compliance
  • Fewer missed classes due to dress code violations
  • Improved academic performance through better attendance
  • Enhanced confidence in personal appearance
  • Real-time feedback and learning
Benefits for University
  • Consistent and objective dress code enforcement
  • Reduced burden on security personnel
  • Better compliance rates through education
  • Improved student-administration relations
  • Data-driven insights on compliance trends
Current Limitations
Known Limitations
  • Limited to female students only
  • Image-based evaluation only (no video)
  • Focuses on clothing only (excludes shoes, makeup, hairstyles)
  • Dataset size constraints affecting accuracy
  • Lighting and image quality dependencies
Future Improvements
  • Extend to male students with separate model
  • Add video analysis capabilities
  • Include comprehensive appearance evaluation
  • Expand dataset for better diversity and accuracy
  • Implement real-time camera integration
